Usage

featureLayerOptions()

addEsriFeatureLayer(map, url, useServiceSymbology = FALSE,
  popupFunction = NULL, popupOptions = leaflet::popupOptions(),
  layerId = NULL, group = NULL, options = featureLayerOptions())

Arguments

map
The leaflet map
url
url of the FeatureService or MapService.
useServiceSymbology
whether to use the symbology set when a service was published.
popupFunction
The Javascript function for popups
popupOptions
The Options for the popup
layerId
Unique ID for the layer
group
The group this layer belongs to.
options
Basemap Layer Options.

Description

Options for featureLayers

Adds an ArcGIS Feature Layer. FeatureLayer is used to visualize, style, query and edit vector geographic data hosted in both ArcGIS Online and published using ArcGIS Server. Copyright text from the service is added to map attribution automatically.
